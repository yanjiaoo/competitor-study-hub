🚀 GitHub部署命令 (用户名: yanjiaoo)

📋 执行以下步骤：

1️⃣ 创建GitHub仓库
   - 访问: https://github.com/new
   - 仓库名: competitor-study-hub
   - 设为 Public
   - 不要勾选 "Add a README file"
   - 点击 "Create repository"

2️⃣ 推送代码到GitHub
   打开命令提示符，复制粘贴以下命令：

   cd C:\Users\yanjiaoo\Desktop\competitor-study-site
   git remote add origin https://github.com/yanjiaoo/competitor-study-hub.git
   git push -u origin main

3️⃣ 启用GitHub Pages
   - 进入 https://github.com/yanjiaoo/competitor-study-hub
   - 点击 Settings 标签
   - 左侧菜单点击 Pages
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "main"
   - 点击 Save

4️⃣ 访问你的网站
   几分钟后网站将在以下地址可用：
   🌐 https://yanjiaoo.github.io/competitor-study-hub
