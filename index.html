    🗣️ 卖家运费相关动态与反馈

    • Amazon官方公告：2026年4月17日起，FBA履约费加收3.5%燃油及物流附加费，平均每件商品增加$0.17。覆盖美国和加拿大FBA、MCF及Buy with Prime。
      Value Added Resource报道 → PPC Land报道 →
    • Unicargo FBA头程报告：2026年1月海运FCL费率：美西$2,000-$4,100/柜，空运约$7/kg，LCL约$118/CBM。费率处于峰后稳定期。
      Unicargo完整报告 →
    • 行业分析：Amazon 2026年FBA费用全面调整，包括履约费、3.5%燃油附加费、入库配置费、月度仓储费、老库存附加费、低库存费、退货处理费等多项变动。
      Goat Consulting费率全解析 →
    • 卖家情绪：多个卖家社区对3.5%燃油附加费表达不满，认为在运费已处低位的情况下加收附加费不合理，叠加广告费支付方式变更（取消信用卡），进一步挤压利润空间。
