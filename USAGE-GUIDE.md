# 竞争对手实时资讯使用指南

## 🎉 新功能概览

您的Amazon Competitor Study Hub现已成功更新，新增了四大竞争对手平台的实时资讯功能：

### 🏪 支持的竞争对手平台
- **Temu** - 拼多多海外版，主打低价商品
- **Shein** - 快时尚电商平台
- **TikTok** - 短视频+电商的社交购物
- **Joybuy** - 京东国际版

## 🚀 主要功能

### 1. 实时资讯更新
- ⏰ **自动刷新**: 每2分钟自动检查新资讯
- 🔄 **手动刷新**: 点击"立即刷新"按钮
- 🔔 **更新通知**: 有新资讯时会显示通知

### 2. 智能过滤系统
- 🏪 **按平台过滤**: Temu、Shein、TikTok、Joybuy
- 📱 **按类型过滤**: 社交媒体、官方网站、新闻发布
- 🔍 **全文搜索**: 搜索标题、内容、标签

### 3. 资讯源管理
- ➕ **添加新源**: 可自定义添加资讯源
- 🗂️ **分类管理**: 按平台和类型组织
- 🔗 **多渠道支持**: 支持Twitter、Instagram、官网、RSS等

## 📖 使用方法

### 查看实时资讯
1. 访问网站：https://yanjiaoo.github.io/amazon-competitor-study-hub/
2. 点击导航栏的"实时资讯"
3. 使用过滤器选择感兴趣的平台或类型
4. 点击"查看原文"跳转到资讯源

### 添加自定义资讯源
1. 点击"管理资讯源"按钮
2. 填写资讯源信息：
   - **名称**: 如"Temu官方LinkedIn"
   - **链接**: 完整的URL地址
   - **类型**: 选择资讯类型
   - **平台**: 选择对应平台
3. 点击"添加资讯源"

### 控制自动更新
- 🟢 **启用自动刷新**: 绿色指示器，每2分钟更新
- 🔴 **暂停自动刷新**: 点击"自动刷新"按钮切换
- ⚡ **立即刷新**: 点击"立即刷新"手动更新

## 🆘 常见问题

**Q: 为什么有些资讯显示"刚刚"？**
A: 这是模拟数据的时间戳，实际部署时会显示真实时间。

**Q: 如何添加其他竞争对手平台？**
A: 可以通过"管理资讯源"添加，或联系开发者扩展平台选项。

**Q: 资讯更新频率可以调整吗？**
A: 可以，通过修改代码中的刷新间隔参数。

**Q: 支持哪些类型的资讯源？**
A: 支持Twitter、Instagram、官网、RSS、新闻网站等多种源。
