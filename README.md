# Competitor Study Hub

## 🚀 快速启动

**Windows用户:**
双击 `start-server.bat` 文件

**Linux/Mac用户:**
运行 `./start-server.sh`

然后在浏览器中访问: http://localhost:8000

## 📁 当前内容

网站已包含以下内容：
- 5个双月度卖家学习报告 (2025年3月-2026年1月)
- 1个多渠道客户回流深度研究

## 🔄 添加新内容

### 方法1: 使用脚本
```bash
./add-content.sh
```

### 方法2: 手动编辑
编辑 `content.json` 文件添加新条目

## 📂 文件结构

```
competitor-study-site/
├── index.html              # 主页面
├── content.json            # 内容数据 (编辑这个文件添加新内容)
├── start-server.bat        # Windows启动脚本
├── start-server.sh         # Linux/Mac启动脚本
├── reports/                # 报告文件夹
│   ├── 2025 SL Seller learning bi-monthly report - March.pdf
│   ├── 2025 SL Seller learning bi-monthly report - May.pdf
```
