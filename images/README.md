# 添加PDF预览图片指南

## 📸 如何添加PDF第一页截图

### 1️⃣ 生成PDF截图
1. 打开PDF文件
2. 截取第一页内容
3. 保存为JPG格式

### 2️⃣ 图片命名规则
- 2026年1月: `2026-jan-preview.jpg`
- 2025年3月: `2025-mar-preview.jpg`
- 2025年5月: `2025-may-preview.jpg`
- 2025年7月: `2025-jul-preview.jpg`
- 2025年10月: `2025-oct-preview.jpg`
- Multi Channel研究: `multi-channel-preview.jpg`

### 3️⃣ 放置位置
将截图文件放入 `images/` 文件夹

### 4️⃣ 图片要求
- 格式: JPG
- 尺寸: 建议宽度800px以上
- 质量: 清晰可读

### 5️⃣ 自动处理
- 如果图片不存在，网站会自动隐藏预览区域
- 图片会自动调整大小适应卡片布局

## 🎯 效果
添加预览图片后，每个Flash和研究报告卡片都会显示PDF第一页的预览，让用户快速了解内容概览。
